## Mobile application dev tools
* Client - JavaScript, React Native
* Server - NodeJS, Express
* Database - MySQL

## Run the application

### Initial Installation
```
npm i --force
```

### Frontend Installation
```
cd Client
npm i --force
```

### Backend Installation
```
cd Server
npm i --force
```

### Run the application (Client)
```
npm start
```
or
```
expo start
```

### Run the application (Server)
```
npm start
```
or
```
nodemon server.js
```
